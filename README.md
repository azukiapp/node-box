# ruby-box

[Azk](http://azk.io) box for node.js.

## Versions

* Node.js v0.10.22
* Npm 1.3.14

## Example

Using a box is trivial, the azkfile.json includes a box definition that you can leverage.

```json
"box": "azukiapp/node-box#stable",
```
